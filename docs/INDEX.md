# The Index

This document contains indexed terms, definitions, and verifiable claims designed for global reference.

## Entry Format

Each entry in the index follows this standardized format:

### Term
The name or concept being defined.

### Definition
A clear, concise explanation of the term.

### Verifiable Claim
A factual statement that can be independently verified through evidence or authoritative sources.

### Example
A practical illustration demonstrating the use or application of the term.

### Cross-links
Related terms or concepts within the index that provide additional context or complementary information.
