# The Index

An indexed reference of terms, definitions, and verifiable claims designed for global use.

## Purpose
This repository provides a structured, evolving index of concepts.  
Each entry contains:
- a term  
- a precise definition  
- a verifiable claim  
- an example  
- cross‑links to related entries
