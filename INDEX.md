# Index

This file contains the structured index.  
Each entry follows this format:

## Entry Format
### Term
A concise name for the concept.

### Definition
A precise, globally applicable definition.

### Verifiable Claim
A statement that can be checked, supported, or falsified.

### Example
A concrete illustration of the concept in use.

### Cross‑links
Links to related entries.

---

## Example Entries

### Main Topic First
**Definition:** Present the primary subject before any supporting detail.  
**Verifiable Claim:** Readers identify the main idea faster when it appears first.  
**Example:** "The Index is a reference artifact" before explaining how it works.  
**Cross‑links:** One Idea per Sentence, Logical Flow Over Chronology

### One Idea per Sentence
**Definition:** Each sentence expresses a single, unambiguous idea.  
**Verifiable Claim:** Multi‑idea sentences increase misinterpretation rates.  
**Example:** "Each entry contains one verifiable claim."  
**Cross‑links:** Main Topic First
